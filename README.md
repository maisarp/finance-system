# Finance System

Este projeto é uma aplicação de backend desenvolvida para gestão financeira pessoal, com o objetivo de substituir controles manuais em planilhas e automatizar cálculos complexos de orçamento.

## O Problema

A aplicação nasce da necessidade de organizar três grandes contextos financeiros:
1. **Despesas Fixas**: Contas de consumo e dívidas de longo prazo.
2. **Gestão de Cartões de Crédito**: Controle de múltiplos emissores, vencimentos e parcelamentos.
3. **Gestão de Terceiros (Repasses)**: Controle de compras realizadas no cartão de crédito pessoal para terceiros, diferenciando o gasto pessoal do que é "a receber".

## Funcionalidades Principais (MVP)

- **Cadastro de Despesas Recorrentes**: Lançamento automático de contas fixas mensais.
- **Motor de Parcelamento**: Geração automática de parcelas futuras a partir de uma compra única.
- **Módulo de Repasse**: Identificação de compras feitas para terceiros para geração de "Extrato de Cobrança" e previsão de reembolsos.
- **Dashboard Unificado**: Visão clara do Saldo Real (Renda - Gastos Fixos - Faturas Pessoais + Reembolsos).

## Objetivo

Fixar conceitos de arquitetura de software, Clean Code e desenvolvimento backend, transformando uma dor real de organização financeira em uma solução robusta e escalável.
